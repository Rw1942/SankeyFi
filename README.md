# SankeyFi

SankeyFi is a fast in-browser Sankey builder powered by DuckDB-Wasm.

The app source lives in `Sankeyfi/`.

## Quick Start

```bash
cd Sankeyfi
npm install
npm run dev
```

Open the local URL shown in the terminal (usually `http://localhost:5173`).
